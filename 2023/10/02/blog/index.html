how to connect the hexo and github for blog writing

first step

download the git(file) and node.js;
note: the path of git need to set in environment variable setting
then install hexo in the cmd;

npm install -g hexo-cli;

second step

initial the blog file for the task;

hexo init blog

note that needing to initial file in blog diraction

npm install
